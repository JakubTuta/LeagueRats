/riot/account/v1/accounts/by-puuid/{puuid}
OPIS: zwraca najważniejsze informacje o koncie, takie jak nazwa użytkownika, tag oraz unikalne PUUID
UŻYCIE W APLIKACJI:
- sprawdzenie czy gracz zmienił nick (puuid jest zawsze stałe, ale gameName i tagLine mogą się różnić)
PATH PARAMS:
- puuid: string
RESPONSE:
- puuid: string
- gameName: string
- tagLine: string


/riot/account/v1/accounts/by-riot-id/{username}/{tag}
OPIS: zwraca najważniejsze informacje o koncie, takie jak nazwa użytkownika, tag oraz unikalne PUUID
UŻYCIE W APLIKACJI:
- podstawowe wyszukiwanie konta
- pozyskanie puuid, który jest podstawowym polem większości zapytań
PATH PARAMS:
- puuid: string
RESPONSE:
- puuid: string
- gameName: string
- tagLine: string


/lol/champion-mastery/v4/champion-masteries/by-puuid/{puuid}
OPIS: zwraca pełną listę postaci zagranych przez użytkownika razem z poziomem i punktami maestrii, czyli poziomem opanowania postaci przez gracza
UŻYCIE W APLIKACJI:
- podanie dodatkowych informacji o koncie
PATH PARAMS:
- puuid: string
RESPONSE:
- championId: int
- championLevel: int
- championPoints: int


/lol/league/v4/entries/by-puuid/{puuid}
OPIS: zwraca informacje o lidze i dywizji gracza, jego pozycji w lidze oraz ilość zwycięstw i porażek
UŻYCIE W APLIKACJI:
- podstawowa informacja o koncie, na podstawie której można poznać swoją pozycję w drabince w porównaniu do innych graczy
PATH PARAMS:
- puuid: string
RESPONSE:
- puuid: string
- leagueId: string
- queueType: string
- tier: string
- rank: string
- leaguePoints: int
- wins: int
- loses: int


/lol/league/v4/challengerleagues/by-queue/{queue_id}
OPIS: zwraca informacje o wszystkich graczach w najwyższej możliwej lidze, czyli pretendencie (challanger)
UŻYCIE W APLIKACJI:
- przygotowanie tabeli z najlepszymi graczami w regionie
PATH PARAMS:
- queue_id: string
RESPONSE:
- gameName: string
- tagLine: string
- puuid: string
- rank: int
- leaguePoints: int
- wins: int
- losses: int
- league: string


/lol/match/v5/matches/by-puuid/{puuid}/ids
OPIS: zwraca id meczów rozegranych przez gracza
UŻYCIE W PROJEKCIE:

- puuid: string
QUERY PARAMS:

- endTime: int
- queue: int
- type: string
- start: int
- count: int
RESPONSE:
- ids: list[string]
